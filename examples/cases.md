# Opus Surface Regression Cases

Use these prompts to check whether Codex keeps the intended Opus-like surface without becoming bloated.

## Analysis

1. Analyze a GitHub repo and explain what it helps with in practice.
2. Compare two architecture options and recommend one with a compact risk matrix.

## Review

3. Review a diff and report findings first, then residual risk.
4. Reconstruct a project's current state from plans, branches, and notes.

## Postmortem

5. Summarize a broken rollout and separate facts, judgment, and next action.
6. Read a design doc and identify the top unresolved decisions.

## Simple Control Cases

7. What is Python?
8. Translate one short sentence.

## Pass Criteria

- Analysis/review/postmortem prompts open with an inspection angle and early verdict.
- Final answers use narrow matrices and grouped judgments.
- Simple control cases stay short.
- No English internal-monologue leak.
- No polite follow-up menu.
